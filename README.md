# 🔥 FIRE Simulator

Plan your journey to Financial Independence with real-life scenarios — not static assumptions.

---

## 🚀 What is this?

FIRE Simulator is an interactive financial planning tool that helps you answer:

* What happens if I switch jobs?
* Can I afford having a child?
* When can I actually retire?
* What if markets underperform?

Unlike traditional FIRE calculators, this simulates your **life over time**, not just a fixed formula.

---

## ✨ Features (MVP)

* 📈 Net worth projection over time
* 💰 Income vs expense modeling
* 🔁 Year-by-year simulation engine
* ⚡ Fast, client-side calculations
* 🧠 Built with extensibility for life events

## ⚙️ Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/your-username/fire-simulator.git
cd fire-simulator
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run locally

```bash
npm run dev
```

Open http://localhost:3000

---

## 🧠 How It Works

At its core, the simulator runs a yearly loop:

* Calculate savings (income - expenses)
* Add to investments
* Apply growth rate
* Repeat over time

This creates a dynamic projection of your financial future.
